# Local PRDs

The **`prd`** skill writes drafts here before implementation:

```text
tasks/prd-<feature-name>.md
```

Use **`to-issues`** to break a PRD into GitHub issues, or **`to-prd`** to publish a synthesized PRD directly to GitHub when you already have enough context in chat.

See [docs/agents/workflow.md](../docs/agents/workflow.md).
