Plataforma SaaS multi-empresa para negocios de servicios: tickets, cobranza, facturación PDF, recordatorios y auditoría — con datos de demostración en vivo.

El problema

Empresas de servicios (HVAC, mantenimiento, instalaciones, facility management) operan con herramientas fragmentadas: hojas de cálculo, WhatsApp, facturas manuales y cero visibilidad multi-sucursal. Eso genera pérdida de ingresos, cobranza inconsistente y cero trazabilidad.

La solución: ZigZag

ZigZag unifica el ciclo completo en una sola plataforma web moderna, multi-tenant y lista para móvil:

Cliente→ Ticket→ Servicios→ Cobro→ Factura PDF
Multi-tenant nativo Aislamiento por empresa con consola operadora central.
Facturación server-side PDF generado en servidor, sin archivos subidos.
RBAC granular Roles Admin, Operator y Viewer con permisos por módulo.
Auditoría enterprise Eventos inmutables en tickets y operaciones sensibles.
Recordatorios Mantenimientos recurrentes con notificaciones in-app.
PWA instalable Acceso móvil desde el navegador, sin app store.

Modelo de negocio objetivo

  • SaaS B2B — Suscripción por empresa (planes starter / standard / enterprise en settings).
  • Franquicias y grupos — Empresa maestra opera N tenants desde una consola.
  • Vertical servicios — HVAC, limpieza industrial, mantenimiento, facility services.
  • Expansión LATAM — UI en español, moneda MXN, RFC y facturación localizable.

Stack tecnológico

Next.js 16 App Router, server actions, API selectiva.
PostgreSQL + Drizzle Migraciones versionadas, multi-tenant explícito.
NextAuth v5 JWT, RBAC, soporte 2FA.
Vercel Deploy con previews por feature, main = producción.
